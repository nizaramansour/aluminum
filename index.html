Conditions & remarks rules

Rules are checked against the current TDS selection; every rule whose Product/Alloy/Temper/Thickness/Width all match gets applied. Rules sharing the same Label combine their Remarks into one row (in rule order) — e.g. a thickness-based Flatness note and a width-based Flatness note can both show together.
